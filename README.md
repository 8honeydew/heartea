# Heartea

Find places through experiences, not keywords.

Heartea is an AI-powered experience search platform that helps users discover places based on the experiences they want rather than relying solely on ratings, categories, or keyword matching.

## Examples

- Quiet cafés for studying
- Restaurants for deep conversations
- Cozy bookstores for browsing
- Remote-work-friendly coffee shops

## Planned Features

- Review analysis
- Experience tagging
- Semantic search
- Consensus detection
- Outlier review detection
- Evidence-backed recommendations
